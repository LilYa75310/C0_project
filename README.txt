# CO_project
Апаратна частина комплексу для вимірювання відносної концентрації оксиду вуглецю у видиху людини.

Main components:
  Мікроконтролер: ESP32.
  Датчик CO: MQ-7.
  MOSFET транзистор: IRLZ44.
  
Getting Started:
  1) Створити фунції обробки значень ppm датчика MQ-7
  2) Реалізувати режим роботи датчкика 
  3) Відкалібрувати датчик MQ-7
  4) Перевірка отриманих значень
